# inchian.top 项目架构说明

> 本文档供维护者查阅，描述站点层级、各目录职责与文件分工。  
> 对外访问者只会看到各二级站点的独立页面，不会看到本说明。

---

## 一、站点层级（单向入口）

```text
┌─────────────────────────────────────────────────────────┐
│  一级入口 · index.html（站点选择页）                        │
│  四个卡片 → 单向进入，不可返回                              │
└──────────┬──────────┬──────────┬──────────┬───────────────┘
           │          │          │          │
           ▼          ▼          ▼          ▼
      studio/    resume/    zichan/   guandian/
      工作室      个人简历    资产分析   个人观点看法
```

### 访问规则

| 规则 | 说明 |
|------|------|
| 一级 → 二级 | 从 `index.html` 点击卡片，使用 `location.replace` 进入，浏览器历史不保留选择页 |
| 不可倒退 | 二级站点内不提供返回一级入口的链接 |
| 站点隔离 | 四个二级站点互不链接、不共用导航栏与样式 |
| 站内导航 | 各二级站点内部页面可自由跳转（如工作室首页 ↔ 低空产业页） |

## 七、guandian/ — 个人观点看法（上市公司研究）

| 文件 | 负责内容 |
|------|----------|
| `index.html` | 公司研究主页：关注列表、深度研究、股价预测、更新记录 |
| `css/site.css` | 琥珀主题样式与公司研究卡片 |
| `js/app.js` | 从数据文件渲染公司列表与研究卡片 |
| `data/companies.json` | 关注公司数据：行情、发展方向、预测、风险 |

### 页面章节

| 锚点 | 内容 |
|------|------|
| `#overview` | 研究说明 |
| `#watchlist` | 关注公司一览（现价、涨跌幅、52 周区间） |
| `#research` | 公司深度研究（逻辑、方向、预测、风险） |
| `#predictions` | 股价与方向预测汇总表 |
| `#log` | 研究更新记录 |

### companies.json 字段说明

| 字段 | 含义 |
|------|------|
| `name` / `ticker` / `market` | 公司名称、代码、市场（A股/港股等） |
| `price.current` / `change` / `range52w` | 二级市场现价、涨跌幅、52 周区间 |
| `direction` | 公司发展方向个人研判 |
| `thesis` | 关注逻辑 |
| `prediction.view` / `target` / `horizon` / `summary` | 观点、目标价、时间窗口、预测摘要 |
| `risks` | 风险与证伪条件 |

## 十、日常更新指引

1. 改完对应二级站点内的 HTML / CSS / JS / 图片  
2. 本地预览：`npx serve -l 8080`，打开 `http://localhost:8080`  
3. 推送至远程仓库后，静态托管自动更新  

部署细节见同目录下的 `部署说明.md`。
