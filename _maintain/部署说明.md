# 部署与更新说明

> 维护者内部文档。站点为纯静态 HTML，无构建步骤。

---

## 更新流程

```text
1. 修改对应二级站点文件
2. 本地预览确认
3. 提交并推送到 main 分支
4. 等待 1～5 分钟，线上自动刷新
```

本地预览：

```powershell
cd 项目根目录
npx serve -l 8080
```

浏览器打开 `http://localhost:8080` 进入一级入口。

---

## 托管说明

- 仓库为公开静态站点，根目录即网站根路径
- `index.html` 为一级入口
- 已配置 `.nojekyll`，静态文件直接托管
- 敏感目录（`_source/`、`素材/` 等）已在 `.gitignore` 中排除，勿手动加入

---

## 自定义域名（可选）

若绑定 `inchian.top`：

- 在托管平台设置自定义域名
- DNS 添加 CNAME 指向托管地址
- 同一域名建议只指向一个平台

---

## 常见问题

| 现象 | 处理 |
|------|------|
| 页面未更新 | 等待几分钟后强制刷新；检查是否推送到 main |
| 图片 404 | 确认文件已提交；路径大小写一致 |
| 一级入口能后退回来 | 确认入口卡片使用 `location.replace` 跳转 |
| 敏感文件泄露 | 检查 `.gitignore`；勿提交 `_source/` |
